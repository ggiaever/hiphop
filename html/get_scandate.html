get_scandate Get scandate from .CEL files

Description

Extracts the scandate from within .CEL files and reformats into YYYY-mm-dd if needed

Usage

get_scandate(celfiles_dir = NULL, cfv = NULL)

Arguments

celfiles_dir

Path to the directory with .CEL files to process

cfv

vector with .CEL file names (give basename with or without .CEL extension)
