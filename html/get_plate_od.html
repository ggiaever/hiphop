get_plate_od {hiphop}R Documentation

get_plate_od

Description

Gets OD data for a given plate from yeast_track database

Usage

get_plate_od(plate_number_in = NULL, plate_name_in = NULL)

Arguments

plate_number_in

The plate number from Yeast Grower, e.g 30285

plate_name_in

The plate name (filename downloaded from Michael's Yeast Grower Desktop Application) 

Here is an example: You can get the same data by using plate number or plate name
> myod1 = get_plate_od(plate_number_in = 30285)
> myod2 = get_plate_od(plate_name_in = "14_07_15_OHC_Media_Replicates_Plate1_20150714_ps1_A_48_T_9.txt")
> all.equal(myod1,myod2) 
[1] TRUE

Value

A data.frame with the following columns:

well: the well number in Yeast Grower plate

sec: time in seconds

od600: optical density at 600 nm (background corrected)

plate_id: the plate number

well_alias: well coordinate in Yeast Grower plate

UserPlateCoord: the coordinates in the user's experimental plate Note: works for 48 well plates ...pending to add when using 96-well plates (in such a case the UserPlateCoord column just shows NAs)
