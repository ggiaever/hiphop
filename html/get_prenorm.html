get_prenorm

Description

Gets prenorm data from a set of .CEL files (using external PERL code).

Usage

get_prenorm(celfiles_dir = NULL, cfv = NULL, getmask = FALSE,
  scriptsdir = NULL)

Arguments

celfiles_dir

Path to the directory with .CEL files to process

cfv

vector with .CEL file names (give basename with or without .CEL extension)

getmask

boolean parameter default FALSE, if set to TRUE the output changes from a data.frame with the .prenorm data to a list with two elements: prenorm and mask.

prenorm contains the data.frame with the prenorm data

mask is a list with elements being data.frames with mask data for each of the .CEL files.

To retrieve the information you can follow: 

cfv = c("@51133100740803060516423465736227","@51133100740803060516423465735925")
y = get_prenorm(cfv = cfv, getmask = T)
names(y)  # prenorm mask
y$prenorm # data.frame with prenorm data
y$mask[,"@51133100740803060516423465736227"]# data.frame with mask data for that chip
scriptsdir

Path for the PERL code needed

Value

If getmask = FALSE, returns a data.frame

If getmask = TRUE, returns a list with two elements:

prenorm: a data.frame

mask: a list with data.frames for each .CEL file used in the input.
