bp_genenames_gmt_20160903 {hiphop}R Documentation

Information for Biological Process GO annotations from SGD as of 2016-09-03

Description

This list is used internally for GO enrichment analysis with function run_go_enrichment The information is composed from the files below (corresponding to 2016-09-03 download) http://downloads.yeastgenome.org/curation/literature/gene_association.sgd.gz http://downloads.yeastgenome.org/curation/chromosomal_feature/SGD_features.tab http://downloads.yeastgenome.org/curation/literature/go_terms.tab

Usage

bp_genenames_gmt_20160903

Format

a list with 2907 elements (vectors of gene names)

list names

The list names are the biological process names
